# Zbrush Zscript

> **主要应用方向是通过Zbrush Zscript文件批量生成并制作模型信息数据集。**<br>


## main210105.TXT文件说明
main210105.TXT能够导出十组不同的模型信息文件组。
我们以模型文件、置换贴图、法线贴图和正面照片四种相关文件为一个模型信息文件组，同时每文件组都包含了43个属性的不同属性值
